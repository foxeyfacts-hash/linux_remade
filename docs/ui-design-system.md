# UI Design System

Goal: Windows 11 style without using Microsoft assets.

## Typography

- Primary: Segoe UI Variable (if available)
- Fallback: Noto Sans, Cantarell, sans-serif
- UI sizes: 11, 12, 14, 16, 20, 24, 32
- Line height: 1.2 to 1.4

## Color tokens

- bg.base: #F3F3F3
- bg.surface: #FFFFFF
- bg.surface.alt: #F7F7F7
- text.primary: #1A1A1A
- text.secondary: #5A5A5A
- accent.primary: #0078D4
- accent.hover: #1A88E6
- accent.pressed: #0063B1
- border.subtle: #E2E2E2
- border.strong: #C8C8C8
- success: #107C10
- warning: #C19C00
- danger: #D13438

## Radii and spacing

- radius.xs: 4
- radius.sm: 6
- radius.md: 10
- radius.lg: 14
- radius.xl: 18
- spacing.1: 4
- spacing.2: 8
- spacing.3: 12
- spacing.4: 16
- spacing.5: 20
- spacing.6: 24
- spacing.7: 32

## Elevation

- shadow.sm: 0 2 8 rgba(0,0,0,0.10)
- shadow.md: 0 8 24 rgba(0,0,0,0.16)
- shadow.lg: 0 16 48 rgba(0,0,0,0.20)

## Effects

- Acrylic blur: 16 to 24px with 0.75 to 0.85 opacity surface overlay.
- Mica background: subtle noise + linear gradient.
- Motion: 160 to 240ms ease-out, 0.9 to 1.0 scale for menus.
